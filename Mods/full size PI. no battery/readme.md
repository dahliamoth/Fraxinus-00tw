Removes the battery and adds mounts for a full size RPI or BTT Pi insted of using a Pi zero 2w.
Print a second set of mounts for the SKR Pico. Rpi has the same mount pattern as SKR Pico.

### Known fitment issues
- For the BTT Pi, I did have to manually modify the mount and space it up on the mount. BTT Pi has some stuff on the underside that interferes.
Unsure if RPI will fit better.
- USB side panel mod interferes. Ended up moving the mount towards the front of machine and using double side tape rather than the screw mount

Never bothered to fix any of it in CAD
