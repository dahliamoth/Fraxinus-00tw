Intended to be used with [Moonraker loader](https://github.com/shiftingtech/Moonraker-loader), the USB ADXL, or for an external USB camera.

### BOM
- USB panel mount https://www.amazon.com/dp/B09SQ86ZJM
- 2x heatsets
- 2x screws (dont remember size)

### Known fitment issue
- Will likely interfere with larger size batteries or if using a the full size PI mod (moving the second PI back and using double side tape for the mount fixed that)
