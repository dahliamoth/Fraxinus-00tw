# Fraxinus 00tw 3MF ファイル

このディレクトリには、Fraxinus 00tw の 3D プリント用 3MF ファイルを収録しています。  
Fraxinus 00tw は、Fraxinus コミュニティによる Fraxinus 00w ベースの派生プロジェクトです。  
詳しくは、このリポジトリの [README.md](../README.md) を参照してください。

このディレクトリ全体は [こちら](https://download-directory.github.io/?url=https%3A%2F%2Fgithub.com%2FFraxinus-3D%2FFraxinus-00tw%2Ftree%2Fmain%2F3MFs) から一括ダウンロードできます。

## 名前の見方

- 丸括弧 `(...)` のディレクトリ  
  いずれかを選んで使う選択式の部品です。たとえば `(normal bed)` と `(heated bed)` は、採用するベッド構成に合わせて選びます。
- 角括弧 `[a]` のファイル  
  アクセントカラーで印刷することを想定した部品です。
- 角括弧 `[a2]` のファイル  
  2つ目のアクセントカラーで印刷することを想定した部品です。
- `_x1` や `_x4` のような末尾  
  必要な印刷数量を示しています。

### How to Read Names

- Directories in parentheses, such as `(...)`:  
  These contain selectable parts. For example, choose either `(normal bed)` or `(heated bed)` depending on the bed configuration you are building.
- Files marked with `[a]`:  
  These parts are intended to be printed in an accent color.
- Files marked with `[a2]`:  
  These parts are intended to be printed in a second accent color.
- Suffixes such as `_x1` or `_x4`:  
  These indicate the required print quantity.
