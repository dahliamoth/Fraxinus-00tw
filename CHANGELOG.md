# Changelog

このファイルには、Fraxinus 00tw の設計データ、ドキュメント、配布物に関する主な変更を記録します。

This file records major changes to Fraxinus 00tw design data, documentation, and distributed files.

## 未リリース / Unreleased

### Added

- ルート README に Highlights / 仕様概要を追加しました。
- BOM への入口として `BOM.md` を追加しました。
- 構成選択やダウンロード支援に使うための `BuildManifest.json` を追加しました。
- `3MFs/README.md` に、ファイル名とディレクトリ名の見方を追加しました。
- `DXFs/README.md` に、パネルデータの概要とクレジットを追加しました。
- `Manuals/README.md` に、組立マニュアルの概要とライセンス説明を追加しました。
- `Firmware/README.md` に、Klipper 設定例とセットアップ参考リンクの入口を追加しました。

### Added (English)

- Added Highlights / specification overview to the root README.
- Added `BOM.md` as the entry point for the BOM.
- Added `BuildManifest.json` for configuration selection and download tooling.
- Added file and directory naming notes to `3MFs/README.md`.
- Added an overview and credits for panel data to `DXFs/README.md`.
- Added an overview and license notes for the assembly manual to `Manuals/README.md`.
- Added `Firmware/README.md` as an entry point for Klipper configuration examples and setup references.

### Changed

- ルート README から BOM、3MF、DXF、PrintSets、Manuals、変更履歴への導線を整理しました。
- ライセンス説明に、許可のない組み立てキット化や販売を認めない旨を追加しました。
- README と LICENSE の CC BY-NC-SA 4.0 対象に BOM を明記しました。
- `DXFs/README.md` のクレジット表記を敬称なしに統一しました。

### Changed (English)

- Organized root README links to the BOM, 3MF files, DXF files, PrintSets, Manuals, and change history.
- Added license notes stating that unauthorized assembly kit production or sales are not permitted.
- Explicitly listed the BOM as covered by CC BY-NC-SA 4.0 in the README and LICENSE.
- Standardized credits in `DXFs/README.md` without honorifics.
