### Fraxinus 00tw Assembly Manual

This directory contains the assembly manual for Fraxinus 00tw.

### File

- `Fraxinus_00tw_manual_rev1.05.pdf`:

### Overview

- Wiring diagrams
- Frame assembly
- Drive system assembly
- Print head and bed area assembly
- Control system assembly
- Door panel, top hat, and exterior assembly
- Selectable configurations:  
  Bed, extruder, control panel, feet, and print head options are covered.
