# Fraxinus 00tw Firmware

このディレクトリには、Fraxinus 00tw 用のファームウェア関連ファイルを収録する予定です。

## PrinterConfig

`PrinterConfig/` には、Klipper 用のサンプル設定ファイルを収録しています。  
これらの設定ファイルは、将来的に Fraxinus 00tw 向けの参考設定として整理して公開する予定です。

## English

### Fraxinus 00tw Firmware

This directory is intended for firmware-related files for Fraxinus 00tw.

### PrinterConfig

`PrinterConfig/` contains sample configuration files for Klipper.  
These files are intended to be organized and published in the future as reference configurations for Fraxinus 00tw.
