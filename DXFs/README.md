# Fraxinus 00tw DXF ファイル

このディレクトリには、Fraxinus 00tw のパネル加工用 DXF ファイルを収録しています。

## ファイル

- `panel back.dxf`  
  背面パネル用のデータです。
- `panel door.dxf`  
  ドアパネル用のデータです。
- `panel side x2.dxf`  
  側面パネル用のデータです。2枚使用します。
- `panel top.dxf`  
  天面パネル用のデータです。

## クレジット

Fraxinus 00tw の設計は [SummerOrange](https://x.com/PlasticMikan) によるものです。  
DXF データの作成は [dahliamoth](https://github.com/dahliamoth) によるものです。

## English

### Fraxinus 00tw DXF Files

This directory contains DXF files for Fraxinus 00tw panel cutting.

### Files

- `panel back.dxf`:  
  Data for the back panel.
- `panel door.dxf`:  
  Data for the door panel.
- `panel side x2.dxf`:  
  Data for the side panels. Use two panels.
- `panel top.dxf`:  
  Data for the top panel.
